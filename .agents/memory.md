# Memoria de Preferencias Visuales

- **No usar emoticonos/iconos de texto:** Nunca utilices emojis o iconos de texto en la interfaz de usuario (por ejemplo: 📂, 💾, 🚀, ⚠️, 🗑️, 📁, ✕). Se ven informales y poco profesionales. En su lugar, importa e implementa siempre iconos vectoriales reales de la biblioteca `lucide-react` para mantener una estética seria, limpia y de calidad de publicación científica.
